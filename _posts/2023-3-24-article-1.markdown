# 問題
vimのbufferを全部クリアできない。

# 原因
vimのbufferを全部クリアするコマンドを知らない。

# 対応
vimのbufferを全部クリアするコマンドを調べる。

# 調査結果

```bash
# vim cliで下記のコマンドを入力する
bufdo bd
```
